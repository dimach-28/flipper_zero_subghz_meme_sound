# 📻 Flipper Zero FM Sound Collection

**Коллекция мемных и популярных звуков для FM-радио / раций**

---

## 🎯 Что это?

Этот репозиторий содержит коллекцию **мемных звуков**, **популярных аудиоклипов** и **смешных шумов**, предназначенных для воспроизведения в эфире на **FM-радио** или **частотах раций**.

Подходит для:
- 🏢 Охранников (розыгрыши по рации)
- 👥 Друзей и коллег (на одной частоте)
- 🎭 Косплея и ролевых игр (добавляет реализма)
- 🎮 Страйбола и пейнтбола (иммерсивное общение)
- 🤣 Просто для смеха (удивляйте своих друзей)

> **⚠️ ВНИМАНИЕ:** Используйте ответственно! Убедитесь, что у вас есть разрешение на передачу на частоте, которую вы используете. Проверьте местные законы и правила перед трансляцией.

---

## 🔧 Сайт для изменения частоты

**Ссылка на сайт:** [https://dimach-28.github.io/flipper_zero_subghz_meme_sound/](https://dimach-28.github.io/flipper_zero_subghz_meme_sound/)

С помощью этого сайта вы можете легко изменить частоту любого файла из этого репозитория.

**Что умеет сайт:**
- 🔄 Менять частоту в любом `.sub` файле
- 📊 Автоматически показывает текущую частоту
- ✏️ Редактировать имя файла перед скачиванием
- 📱 Удобный интерфейс для телефона
- 🎯 Автоматически форматирует частоту (например, `433.92` → `433920000`)

**Как пользоваться:**
1. Загрузите ваш `.sub` файл
2. Введите новую частоту в МГц (например, `440.10`)
3. Нажмите "Apply" — файл обновится
4. Скачайте готовый файл

Это очень удобно, если вы узнали частоту рации и хотите быстро изменить её в файле.

## 📂 Формат файлов

Все файлы в формате **Flipper Zero RAW**:

```

Filetype: Flipper SubGhz RAW File
Version: 1
Frequency: 433920000
Preset: FuriHalSubGhzPreset2FSKDev238Async
Protocol: RAW
RAW_Data: 805 -172 57 -86 ...

```


Сайт для изменения частоты работает именно с этим форматом, заменяя строку `Frequency:`.

---

## 🚀 Как использовать

### Со звуками из репозитория:
1. Скачайте нужный `.sub` файл
2. Если нужно — измените частоту через сайт
3. Загрузите файл на Flipper Zero

### Использование с Flipper Zero:
1. Скопируйте `.sub` файлы в `SD Card/subghz/` на вашем Flipper
2. Перейдите в **Sub-GHz → Saved**
3. Выберите файл и нажмите **Send**

## ❓ Часто задаваемые вопросы

**Вопрос:** Можно ли использовать эти звуки на реальных рациях?  
**Ответ:** Да, но только на частотах, на которые у вас есть разрешение. Всегда проверяйте местные законы.

**Вопрос:** Как узнать частоту рации?  
**Ответ:** Используйте частотомер или посмотрите в инструкции к устройству. Некоторые приложения тоже помогают.

**Вопрос:** Могу ли я добавить свои звуки?  
**Ответ:** Конечно! Создавайте `.sub` файлы с помощью Flipper Zero или других инструментов и добавляйте их в любую папку.

**Вопрос:** Почему мой файл скачивается как .txt?  
**Ответ:** Сайт сохраняет файлы как `.sub` автоматически. Если браузер добавляет `.txt`, просто переименуйте файл вручную.
